# Status

CoroditeP2P is currently in an early public repository stage.

This repository is being prepared as the open project base for:

- protocol specification
- reference implementation
- state and transition semantics
- multi-node testing
- field-study preparation
- public technical documentation

Some earlier internal notes and documents are currently being reconstructed.

The current focus is on:

1. establishing the public repository structure
2. rebuilding the core documentation
3. defining the protocol and peer-state model
4. preparing reproducible test and field-validation setups

This means the repository may still look sparse while the public structure is being rebuilt.
